# Кратко и понятно — Telegram Channel Bot

**Кратко и понятно Bot** — Telegram-бот для канала, через которого пользователи могут отправлять фото, видео, идеи и обращения в поддержку. Администратор получает материалы на проверку, может одобрить или отклонить пост, а после одобрения бот публикует его в канал.

## Кратко о проекте

Проект сделан в одном Python-файле и работает на библиотеке `python-telegram-bot`.

Бот помогает автоматизировать работу Telegram-канала: пользователи отправляют контент, выбирают анонимность, добавляют цитату или подпись, а администратор принимает решение — публиковать пост или нет.

## Что умеет бот

- запуск через команду `/start`;
- главное меню с кнопками;
- просмотр профиля пользователя;
- отправка фото или видео на модерацию;
- добавление цитаты к посту;
- выбор анонимной или открытой публикации;
- отправка поста администратору;
- кнопки для админа: `Одобрить` и `Отклонить`;
- публикация одобренного поста в канал;
- уведомление пользователя о результате;
- обращение в поддержку;
- ответ администратора пользователю;
- кнопка отмены действия;
- реакции на сердечки.

## Для чего нужен проект

Бот нужен для канала **«Кратко и понятно»**, чтобы пользователи могли удобно предлагать посты, фотографии, видео и идеи, а администратор мог быстро проверять материалы перед публикацией.

Весь поток контента проходит через одного бота:

1. Пользователь отправляет материал.
2. Бот спрашивает подпись или цитату.
3. Пользователь выбирает анонимность.
4. Бот отправляет пост админу.
5. Админ нажимает `Одобрить` или `Отклонить`.
6. Если пост одобрен — он публикуется в канал.

## Как запустить

### 1. Установить Python

Нужен Python версии 3.10 или выше.

Проверить Python можно командой:

```bash
python --version
```

### 2. Установить зависимости

В папке с проектом выполните:

```bash
pip install python-telegram-bot
```

### 3. Настроить бота

Откройте файл `bot_kt.py` и заполните настройки:

```python
TOKEN = "YOUR_BOT_TOKEN"
ADMIN_USER_ID = 123456789
ADMIN_USERNAME = "your_admin_username"
CHANNEL_USERNAME = "@your_channel_username"
```

Что нужно заменить:

```text
TOKEN — токен бота от @BotFather
ADMIN_USER_ID — Telegram ID администратора
ADMIN_USERNAME — username администратора
CHANNEL_USERNAME — username канала, куда бот будет публиковать посты
```

Важно: бот должен быть добавлен в канал и иметь права администратора, чтобы публиковать посты.

### 4. Запустить бота

```bash
python bot_kt.py
```

После запуска откройте Telegram и напишите боту:

```text
/start
```

## Основные кнопки бота

### Профиль

Кнопка:

```text
👤  Профиль  🖼️
```

Показывает информацию о пользователе:

- ID;
- username;
- дату присоединения;
- должность.

### Поддержка

Кнопка:

```text
❓  Поддержка  🤝
```

Пользователь описывает проблему, а бот отправляет сообщение администратору. Администратор может ответить пользователю или закрыть заявку.

### Отправить пост

Кнопка:

```text
📤  Отправить Пост  🖌️
```

Пользователь отправляет фото или видео, добавляет цитату и выбирает, публиковаться анонимно или нет.

### Предложить идею

Кнопка:

```text
💡  Предложить Идею  🌟
```

Бот показывает контакт администратора, чтобы пользователь мог предложить идею напрямую.

## Демо: отправка поста

```text
Пользователь нажимает:
📤 Отправить Пост

Бот отвечает:
Загрузите ваше фото или видео.

Пользователь отправляет фото.

Бот спрашивает:
Добавьте цитату или напишите "-".

Пользователь пишет:
Живи моментом.

Бот спрашивает:
Хотите сохранить анонимность?

Пользователь выбирает:
Да

Администратор получает пост с кнопками:
✅ Одобрить
🚫 Отклонить

Если админ нажимает ✅ Одобрить:
бот публикует пост в канал.
```

## Демо: поддержка

```text
Пользователь нажимает:
❓ Поддержка

Бот просит:
Подробно опишите, что случилось.

Пользователь пишет:
У меня не отправляется видео.

Администратор получает сообщение и нажимает:
🔍 Уточнить

Администратор пишет ответ.

Пользователь получает сообщение от поддержки.
```

## Демо: отклонение поста

```text
Пользователь отправляет фото или видео.

Администратор получает пост на проверку.

Администратор нажимает:
🚫 Отклонить

Пользователь получает сообщение:
Нам очень жаль, но ваше сообщение отклонено.
```
